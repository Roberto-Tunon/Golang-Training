Cuando se crea un Repo nuevo hay que ejecutar los siguientes comandos en Git:
-   git init
-   touch temp.txt
-   git add temp.txt
-   git commit -m "First"
-   git branch -M main
-   git remote add origin git@github.com:Roberto-Tunon/GO-Barcelona-Training.git
-   git push -u origin main

Para inicializarlo en GO hay que ejecutar:
go mod init github.com/Roberto-Tunon/GO-Barcelona-Training
